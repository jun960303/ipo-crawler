# 📈 IPO Crawler (공모주 자동 크롤러)

Python 기반으로 제작한 **공모주 정보 자동 크롤링 + 조회 + 엑셀 내보내기 솔루션**입니다.  
사용자가 실행하기만 하면 주요 증권사 / 공모주 일정 / 수요예측 / 상장일정을 자동 수집하여  
데이터베이스에 저장하고 깔끔하게 출력합니다.

이 프로젝트는 Python 자동화 공부와 가족을 위한 실사용 툴 제작을 목표로 개발되었습니다.

---

## 🚀 주요 기능

### 🔍 1. 공모주 정보 자동 크롤링
- 38커뮤니케이션 / 주요 증권사 공모주 일정 크롤링
- 수요예측 일정, 청약일, 상장일 등 자동 수집
- 중복 데이터 방지 알고리즘 적용

### 📊 2. DB 저장 (SQLite)
- 구조화된 스케줄 테이블  
- 중복 체크 후 새로운 데이터만 저장
- `ipo.db` 파일로 로컬에 유지

### 🖥 3. GUI 기반 사용자 인터페이스
- 버튼 클릭만으로 크롤링 / 조회 / 엑셀 내보내기 가능
- 로그창에서 모든 작업 확인 가능  
- 초보자도 쉽게 실행할 수 있도록 직관적 인터페이스 제공

### 📁 4. 엑셀(xlsx) 내보내기
- 자동 정렬 + 날짜 형식 + 금액 서식
- 열 너비 자동 조정
- 헤더 스타일(파란색 배경 + 흰색 글자) 적용
- 바탕화면에 자동 저장

### 🐶 5. 강아지 아이콘이 적용된 EXE 실행 파일
- PyInstaller로 제작  
- Windows 환경에서 Python 설치 없이 바로 실행  
- 엄마도 사용하기 쉽게 단일 파일 형태로 제공

## 🧱 기술 스택

- **Python 3.10+**
- **Tkinter** – GUI
- **Requests / BeautifulSoup** – 웹 크롤링
- **SQLite3** – 로컬 데이터베이스
- **OpenPyXL** – 엑셀(.xlsx) 생성 / 서식 적용
- **PyInstaller** – EXE 빌드

---

## ▶ 실행 방법 (개발자용)

### **1) 패키지 설치**
```bash
pip install -r requirements.txt

2) 실행
python main.py

3) EXE 생성
pyinstaller --noconsole --onefile --icon=dog_icon.ico --add-data "db/ipo.db;db" main.py

📦 EXE 파일 사용법 (사용자용)

1. ipo_공모주크롤링.exe 파일 실행

2. 버튼 클릭:

크롤링 실행 → 최신 공모주 데이터 자동 수집

전체 예정 공모주 조회

엑셀로 내보내기

3. 엑셀 파일은 자동으로 바탕화면에 저장됩니다.
